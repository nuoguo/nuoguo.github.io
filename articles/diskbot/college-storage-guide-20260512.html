作为一名 Mac 用户 + 穷学生，我找到了最好的免费清理方案。

一个大学生的存储焦虑

大三那年，我买了人生第一台 MacBook Air。256GB 版本，教育优惠后 7999，已经是我能承受的极限。

前半年一切美好。直到某天打开 Final Cut Pro 准备剪视频，系统弹窗：

"启动磁盘几乎已满。"

我懵了。我明明没存多少东西啊？

打开「关于本机」→「存储空间」，看到那个让人绝望的进度条：

  • 系统数据：68GB
  • 应用程序：42GB
  • 文档：53GB
  • 其他：37GB

总共才 200GB 出头，我的文件在哪？那 68GB"系统数据"到底是什么？

我兴冲冲下载完，扫描完磁盘，然后看到了价格：

298 块，够我在食堂吃两周了。

第二次尝试：找"绿色版"

不想花钱，那就找破解版。

我在各种软件站下载了"CleanMyMac X 破解版"，结果：

  1. 安装包带病毒警告
  2. 安装后系统变得卡顿
  3. 某次更新后直接崩溃

得不偿失。 为了省 298 块，我把系统搞坏了，最后还去 Genius Bar 花了一下午重装系统。

▲ 破解软件的血泪教训

转机：发现 DiskBot

大四上学期，我在 GitHub 上偶然发现了 DiskBot。

这是一个完全免费的 Mac 磁盘分析工具，开源，无广告，无付费墙。

我抱着试试看的心态下载了，结果让我惊呆了：

5 分钟，它帮我找到了 32GB 可以安全删除的文件。

DiskBot 帮我找到了什么？

让我来复盘一下 DiskBot 的扫描结果：

1. Xcode DerivedData（12GB）

~/Library/Developer/Xcode/DerivedData/

我大二学 Swift 时留下的编译缓存，早就不用的项目，缓存还在占空间。

2. 微信文件传输助手缓存（8GB）

~/Library/Containers/com.tencent.xinWeChat/

大学四年，和同学的作业、社团文件、表情包，全部存在本地。

3. 下载文件夹的大文件（6GB）

~/Downloads/
  • pr2019_final_v3.mp4 — 大二 PR 作业，2.3GB
  • MATLAB_R2020b.dmg — 早卸载了，安装包还在
  • Keynote_毕业答辩.ppt — 答辩结束了，文件还在

4. 系统日志和崩溃报告（4GB）

~/Library/Logs/
/private/var/log/

系统运行的日志文件，普通人根本不需要。

32GB，说删就删了。我的 Mac 瞬间呼吸通畅。

为什么 DiskBot 适合大学生？

作为过来人，我总结了几点：

1. 完全免费

不用破解，不用找盗版，不用承担病毒风险。关注公众号「诺果AI」就能免费使用全部功能。

对于每个月生活费有限的大学生来说，这太重要了。

2. 可视化，不用学命令行

你不需要会 du、ncdu 这些命令行工具。DiskBot 的 TreeMap 视图，像看地图一样看磁盘占用。

3. 安全，不会误删系统文件

DiskBot 会标注每个文件/目录的安全等级：

  • 🟢 安全 — 可以放心删除
  • 🟡 谨慎 — 删除后可能需要重新下载
  • 🔴 危险 — 系统关键文件，不要动

作为不懂技术的大学生，这个功能太友好了。

4. 原生 App，不占资源

SwiftUI 原生开发，不像 CleanMyMac 那样后台常驻占内存。需要时才打开，用完就关。

我的清理 SOP（标准操作流程）

经过一年多的使用，我总结了一套适合大学生的 Mac 清理流程：

每周一次（5 分钟）：

  1. 打开 DiskBot，扫描主磁盘
  2. 查看"大文件"列表（> 100MB）
  3. 删除已经不需要的安装包、视频、压缩包

每月一次（15 分钟）：

  1. 清理 Xcode DerivedData（如果你不写代码，直接全删）
  2. 清理 ~/Library/Caches/ 中的应用缓存
  3. 清理"下载"文件夹
  4. 清空废纸篓

每学期一次（30 分钟）：

  1. 用 DiskBot 的"重复文件检测"功能
  2. 找出重复的作业文件、社团照片
  3. 清理微信/QQ 文件传输助手的历史文件

坚持这个流程，我的 256GB MacBook Air 用了 4 年，从来没换过。

给大学生的建议

如果你也是一名 Mac 用户 + 穷学生，我的建议是：

  1. 不要买盗版/破解软件 — 风险太大，得不偿失
  2. 不要盲目删系统文件 — 用专业工具，别自己瞎折腾
  3. 定期清理，不要等红了再救急 — 养成习惯，256GB 够用
  4. 关注「诺果AI」公众号 — 免费获取 DiskBot 激活码，还有更多 Mac 使用技巧
